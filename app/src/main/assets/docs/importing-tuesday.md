# Importing Tuesday JS

MIYO includes a partial Tuesday JS importer.

Imported concepts:

- project title and language list
- global variables
- blocks and graph positions
- scenes and background colors
- dialogue text and speaker references
- choices and next-block links
- basic layered art from scene/dialog references where possible

Not automatically preserved as first-class runtime features:

- arbitrary HTML
- arbitrary JavaScript
- plugin-specific runtime behaviors

Unsupported content is stored as raw notes so it can be reviewed manually.
