# Getting Started

MIYO is organized around a typed project model.

## Authoring modes

- Easy mode: graph, scene, dialogue, variable, and preview tools
- Manual mode: file tree, raw JSON view, and Lua script editing

## Project flow

1. Start from a template.
2. Arrange blocks in the graph.
3. Build scenes from layered backgrounds and sprites.
4. Write dialogue pages and choices.
5. Preview from any block.
6. Export a ZIP archive or push a backup to Supabase.

## Runtime notes

- Layers can be persistent across all pages or attached to a specific page.
- Text interpolation supports `<variable_name>` syntax.
- Choices can mutate variables before routing to the next block.
